The Complete Robot Tag Guide

Nov 16, 2016 • Robert Edwardes

4 min read

Meta robots tags have been around for a while. During that time thought Search engines not only have slowly been adding features to them, but they have also expanded beyond the meta tag and robots.txt. This page lists an overview of all the different values you can use in your tag. The official syntax of the meta robot tag:

< meta name="robots" content="value1, value2" />

Which Search Engine Support Which Meat Tags Values

What each Meta Robots Meta Value do

all

  • There are no restrictions for indexing or serving. Note: this directive is the default value and has no effect if explicitly listed.

index

  • Allow search engines robots to index the page, you don’t have to add this to your pages, as it’s the default.

noindex

  • Do not show this page in search results and do not show a “Cached” link in search results.

follow

  • Tells the search engines robots to follow the links on the page, whether it can index it or not.

nofollow

  • Do not follow the links on this page

none

  • Equivalent to noindex, nofollow

noarchive

  • Do not show a “Cached” link in search results.

nocache

  • Same as noarchive, but only used by MSN/Live.

nosnippet

  • Do not show a snippet in the search results for this page

noodp

  • Do not use metadata from the Open Directory project for titles or snippets shown for this page.

notranslate

  • Do not offer translation of this page in search results.

noimageindex

  • Do not index images on this page, but id the image is linked to directly from elsewhere Google will still index them, so using an X-Robot-Tag HTTP header is better.

unavailable_after: [RFC-850 date/time]

  • Do not show this page in search results after the specified date/time. The date/time must be specified in the RFC 850 format.
